

# Silas Wideman – Personal Portfolio Website

A single-page, fully static portfolio site with a **friendly and approachable** design — soft colors, rounded elements, and a welcoming personality that matches Silas's strengths in mentorship, teamwork, and hands-on work.

---

## Design Direction
- **Color palette**: Soft warm tones — creamy whites, gentle greens, warm tans, with a friendly accent color (e.g., soft teal or warm orange)
- **Typography**: Rounded, friendly fonts — approachable headings with clean body text
- **UI touches**: Rounded cards, subtle shadows, gentle hover animations, emoji/icon accents where appropriate
- **Fully static**: No backend needed — deploys directly to Cloudflare Pages or Netlify

---

## Sections (single scrolling page)

### 1. Hero Section
- Large friendly greeting: "Hi, I'm Silas Wideman"
- Short tagline from his summary (dependable, team player, positive attitude)
- Location badge: Tulsa, OK
- Subtle scroll-down indicator

### 2. About Me
- A warm summary paragraph drawn from his resume summary
- Key personal qualities highlighted as friendly badges/chips (e.g., "Team Player", "Dependable", "Positive Attitude")

### 3. Skills
- Skills organized into visual card groups matching his resume categories:
  - Grounds & Animal Care
  - Youth Mentorship
  - General Skills
  - Janitorial & Maintenance
  - Logistics & Processing
- Each category as a rounded card with relevant icon and skill tags

### 4. Experience Timeline
- A clean vertical timeline showing his work history
- Each entry shows: role, organization, location, dates, and key bullet points
- Visual distinction between paid roles, volunteer work, and vocational training

### 5. Education
- Simple card showing Sapulpa High School diploma (2023)

### 6. Contact / Footer
- Display email and phone number with clickable links (mailto / tel)
- Location shown
- Clean, simple footer with a friendly sign-off

---

## Technical Notes
- Fully static React + Vite build — no API calls, no backend
- Responsive design (mobile-first)
- Smooth scroll navigation with a sticky top nav bar
- Ready to deploy with `npm run build` → static output
